# 项目运行：

### 克隆到本地:	
	
	git clone https://github.com/lliihao/funny_quiz.git

### 进入文件夹:		

	cd funny_quiz

### 安装依赖:

	npm install

### 在本地主机上加载服务（localhost:8888）:	
	
	npm run dev

### 开启浏览器查看：	
	localhost:8888

### 发布:	
	npm run build
